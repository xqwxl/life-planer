# Планер жизни — установка на телефон

Это PWA-приложение: один сайт, который ставится как обычное приложение
на iPhone и Android, работает офлайн и хранит данные на устройстве.

Чтобы появился значок на экране и полноэкранный режим, приложение нужно
открыть по https-ссылке (просто открыть файл с телефона — не даст установку).
Ниже — самый быстрый способ это сделать бесплатно.

## Шаг 1. Выложить папку в интернет (1 минута, бесплатно)

Вариант А — Netlify Drop (проще всего, без регистрации):
1. На компьютере открой https://app.netlify.com/drop
2. Перетащи туда ВСЮ папку life-planner (или zip-архив).
3. Через пару секунд получишь ссылку вида https://что-то.netlify.app
4. Эта ссылка и есть твоё приложение. Открой её на телефоне.

Вариант Б — GitHub Pages / Vercel / Cloudflare Pages — если уже пользуешься.
Главное: файлы index.html, manifest.webmanifest, sw.js и иконки лежат рядом,
в одной папке, и открываются по адресу .../index.html

## Шаг 2. Установить на iPhone (Safari)

1. Открой ссылку в Safari (именно Safari, не Chrome).
2. Нажми кнопку «Поделиться» (квадрат со стрелкой вверх).
3. Выбери «На экран Домой».
4. Готово — появится иконка «Планер». Запускается на весь экран.

## Шаг 2. Установить на Android (Chrome)

1. Открой ссылку в Chrome.
2. Меню (три точки) → «Установить приложение» / «Добавить на главный экран».
   (Иногда баннер «Установить» появляется сам внизу.)
3. Готово — иконка на рабочем столе, запуск как у обычного приложения.

## Перенос данных телефон ↔ компьютер

Пока облачной синхронизации нет, перенос — вручную через JSON:
- Настройки → Экспорт: сохранит файл с твоими данными.
- Настройки → Импорт: загрузит этот файл на другом устройстве.

## Что внутри
- index.html — само приложение (всё в одном файле)
- manifest.webmanifest — описание PWA (имя, иконки, цвет)
- sw.js — офлайн-режим (кэширование)
- icon-192.png / icon-512.png — иконки приложения
